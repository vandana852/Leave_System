Employee Leave & Attendance System (PHP + MySQL)
Features:
- Admin and Staff roles (single login page)
- Staff: apply leave, check-in, check-out, view attendance history
- Admin: view/approve/reject leaves, view employees, view attendance records
- Bootstrap sidebar layout (blue theme) for admin and staff as requested
Setup:
1. Create a MySQL database (e.g., leave_system).
2. Import database.sql
3. Update config.php with DB credentials.
4. Place files in your PHP server (htdocs or www).
Default credentials (for demo):
- Admin: admin@example.com / admin123
- Staff: staff1@example.com / staff123
NOTE: This is a demo project. Replace MD5 with password_hash in production.
